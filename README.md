# SafeBlock

PWA mobile-first de prévention, médiation communautaire et mise en sécurité.

## Lancer l'application

```bash
cd /Users/admin/Documents/Codex/2026-07-02/pre/work/safeblock && python3 -m http.server 4173 --bind 127.0.0.1
```

Ouvrir ensuite `http://127.0.0.1:4173`.

## Ecrans inclus

- Authentification, inscription, choix du role et onboarding
- Accueil, carte, heatmap, signalement, validation et intervention
- Canaux, chat, appel audio, visio et notifications
- SOS, itineraire securise, profil, espace mediateur et administration
